Researches

Inverse modeling with user simulators

Inferring the free parameters of the user simulator from observed user behaviors remains challenging. For simulation-based inference, the optimization of the new action policy of the simulated agent is required whenever the free parameters change, but this is computationally impractical. In this study, we introduce a network modulation technique that can obtain a generalized policy that immediately adapts to the given free parameters. We demonstrated that the proposed technique improves the efficiency of user simulator-based inference by eliminating the need to obtain an action policy for novel free parameters.

In CHI 2022 (to appear): Video Project Page

Few-shot adaptation of user behavior model

Behavioral differences across various users are crucial factors that limit the prediction of human motion. Although recent neural network-based modeling methods have improved their prediction accuracy, most did not consider an effective adaptations to different users, thereby employing the same model parameters for all users. To deal with this challenge, we introduce a meta-learning framework to facilitate the rapid adaptation of the model to unseen users.

In IEEE RA-L: Link arXiv

Deep learning-based haptic guidance

Haptic guidance (HG) improves users‚Äô task performance through physical interaction between robots and users. We presented two types of HG: optimal action-based HG, which assists users with an optimal action, user prediction-based HG, which assists users with their intended action, and proposed a combined HG approach which utilizes the both HG types. We applied self-play-based reinforcement learning and meta-learning methods for implementing them. We investigated how the assisting performance and users' subjective evaluation differ according to each HG type.

In CHI 2021: Video Link arXiv

Imaginary rollout-based robotic guide training

Training a robot that engages with humans is challenging, because it is expensive to involve humans in a robot training process requiring numerous data samples. We proposed a deep learning-based model that predict human path following a robot and an evolution strategy-baesd robot training method using "imaginary rollouts" generated by the human predictive model, which compensates for this sample inefficiency problem. With the proposed method, we trained a robotic guide that assists a user to walk along a given path.

In preprint: arXiv

Effect of haptic information in human multitasking

We implemented a multimodal task interruption environment involving the simultaneous presentation of visual information and haptic stimuli in order to investigate how the combined stimuli affect the performance on the primary task (i.e., cost of interruption). A user test (n=21) indicated that, within a visuo-tactile task environment, redundant haptic information may not only increase accuracy on the primary task but also reduce the cost of interruption in terms of accuracy.
